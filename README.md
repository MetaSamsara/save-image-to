# Save image to...

A Firefox extension that adds a right-click context menu to save images into custom subfolders (under Downloads), with smart naming rules and per-folder overrides.

Created by **MetaSamsara** with the help of **ChatGPT 5.2**.

## Features
- Right-click image → Save image to...
- Save into custom folders (subfolders under Downloads)
- Smart naming rules (domain/date/original/etc.)
- Per-folder naming overrides
- Works on tricky sites (Instagram / Pinterest)

## Install (temporary / dev)
1. Firefox → `about:debugging`
2. This Firefox → **Load Temporary Add-on**
3. Select `manifest.json`
