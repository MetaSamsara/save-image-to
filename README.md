# Save image to...

A Firefox extension that adds a right-click context menu to save images into custom folders (subfolders under Downloads), with smart naming rules and per-folder overrides.

Created by **MetaSamsara** with the help of **ChatGPT 5.2**.

## Features
- Right-click image → **Save image to...**
- Multiple custom folder entries
- Smart naming: **site domain + date/time + original filename**
- Optional per-folder naming overrides

---

## Install (temporary / dev)
1. Firefox → `about:debugging`
2. This Firefox → **Load Temporary Add-on**
3. Select `manifest.json`

---

## Install (Firefox Add-ons / AMO)
✅ Install the official signed version here:

**https://addons.mozilla.org/en-US/firefox/addon/save-image-to/**

---

## Notes
Firefox extensions can only save files inside subfolders of your default Downloads directory (WebExtension limitation).
